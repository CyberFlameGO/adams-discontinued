\chapter{OpenIMAJ}
OpenIMAJ\cite{openimaj} is an award-winning set of libraries and tools for
multimedia content analysis and content generation. OpenIMAJ is very broad
and contains everything from state-of-the-art computer vision (e.g. SIFT
descriptors, salient region detection, face detection, etc.) and advanced
data clustering, through to software that performs analysis on the content,
layout and structure of webpages.
